# Resume

Drop your resume file here. Both formats are supported:

- `Your_Name_Resume.pdf` — preferred (attached to outreach emails as-is)
- `Your_Name_Resume.txt` — plain-text fallback (used for ATS scoring if no PDF found)

The scorer and email builder auto-detect the first `.pdf` or `.txt` file in this directory.
Rename your file however you like — no hardcoded filename required.

> **This directory is gitignored.** Your resume is never committed.
